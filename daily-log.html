Your day, on the record.

Daily Log keeps three things in one calm place — and it all lives in a file that stays yours. No account, no cloud, no subscription.

  • Habits — check them off, build streaks, watch the record fill in.
  • Tasks — a simple board for each day: to do, doing, done.
  • Long term — the big goals, fed by the small daily tasks you link to them.
Keep a copy you own. Use Save to file at the bottom — it auto-saves as you go. You can also Install / Add to Home Screen for a full-screen, app-like Daily Log.

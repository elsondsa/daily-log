Welcome. Let's get you set up.

This takes about a minute. Daily Log is a single file — there's nothing to install.

1

Open the app

Double-click daily-log.html to open it in your browser. For the best experience — including one-click Save to file — use Google Chrome or Microsoft Edge on a computer.

Open Daily Log →

2

Keep the file in one folder

Move daily-log.html to a folder you'll keep — say Documents/Daily Log. Open it from that same spot each time, in the same browser, so your history stays put.

3

Turn on “Save to file” (Chrome / Edge)

Scroll to the bottom of the app and click ⤓ Save to file. Choose a name like daily-log-data.json and save it next to the app. From then on, Daily Log auto-saves your data into that file as you work — a real copy you own.

Backups — never lose your data

  • •Save to file keeps a live, auto-saving copy (Chrome/Edge).
  • •Export (bottom of the app) downloads a dated backup .json anytime — works in every browser.
  • •Import restores from any backup, on any device.
4

Install it like an app (optional)

Want it full-screen and one tap away?

Add to Home Screen / Install

  • ▸Desktop Chrome / Edge: open the app from a web address (or a local server), then click the install icon in the address bar.
  • ▸Android (Chrome): menu ⋮ → Add to Home screen.
  • ▸iPhone / iPad (Safari): Share ↑ → Add to Home Screen.
Good to know: Daily Log works fully offline and keeps your data on your device only. Nothing is sent to any server — so your backups are yours to keep.

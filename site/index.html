Good questions

What do I actually get? +
A single daily-log.html file (plus a short start-here guide). Open it in your browser — that's the whole app. No install, no account, no setup.
Which browser do I need? +
Any modern browser works. For the one-click Save to file auto-save, use Chrome or Edge (desktop). On other browsers you get the same app with manual Export/Import for backups.
Does it work offline? +
Yes — completely. It's just a local file, so there's nothing to load from the internet. You can even add it to your home screen and use it like an app.
Where is my data stored? +
On your device only. It lives in your browser's local storage and, if you choose, in a .json file you keep. We never see it — there is no server. Back it up by keeping the file or using Export.
Is this a subscription? +
No. One payment, yours forever. No recurring charges, no upsells.
